eodash-docs ​

Central point for documentation related to the eodash ecosystem — a federated system for Earth Observation data visualization based on STAC.

Key Features ​

  • STAC-Native: Built entirely around SpatioTemporal Asset Catalogs for data management.
  • Modular Architecture: Core framework with extensible widgets and state management.
  • Configurable UI: Easily customizable themes and dashboard layouts.
  • Storytelling: Integrated interactive and guided experiences for EO data.
  • Agnostic Integration: Works as a Web Component or a Vite-based package.

Ecosystem Components ​

The eodash ecosystem consists of three main pillars:

  1. eodash: The core web client dashboard library (Vue 3).
  2. eodash_catalog: A Python helper for generating static STAC catalogs from various EO endpoints.
  3. storytelling: A component for creating interactive, markdown-based guided stories.

Local Development ​

To run the documentation locally, follow these steps:

1. Install dependencies ​

bash
npm install

2. Start the development server ​

bash
npm run docs:dev

The documentation will be available at http://localhost:5173.

3. Build & Preview ​

bash
# Build for production
npm run docs:build

# Preview the production build
npm run docs:preview
